# Gladymar APP
> Warehouses and sales application, developed to give better control of warehouses integrated with a sales and purchases module.
## ✨ Features
- Modules of Users and Sales.
- Detail management of inputs, outputs, warehouses, agencies, products, suppliers, sales, purchases and selling channels.
- Administrative site provided by Django framework.
- Management of users, including permissions, activation and roles.
- Reports generated for outputs, inputs, sales, purchases, and stock of the products.
## 🛠️ Technologies
- Django (DRF)
- React
- PostgreSQL
- Docker
- Vite
- CSS
- HTML5
## 📦 Installation
- Prerequisites
  - Node v24.13.0
- Backend
  - In the backend directory run **docker compose build**
  - Run command **docker compose run** to run the server.
- Frontend
  - Run command **npm install**.
  - Run command **npm run dev**
